# Trabalho-Entropia
Trabalho da aula do Carlson sobre entropia
Instruções:
Considerando o vídeo deste link: https://www.youtube.com/watch?si=UcZsXIkAlMwS8apn&v=PPWEmjnqsF8&feature=youtu.be, o projeto consiste em desenvolver um simulador do “carro” para gerar os dados estatísticos que corroboram os resultados discutidos no vídeo. Não é objetivo principal nem uma implementação detalhada de um carro nem sua visualização. O objetivo é produzir dados para fins de análise estatística baseada em simulações (que são aproximações da realidade, nesse caso, nem tão próximas, mas servem muito bem como exercício de programação).
Requisitos e funcionalidades do projeto:
1) Pensar em estruturas de dados para dar suporte à simulação, em especial o espaço do motor e do ambiente.
2) Implementar a movimentação aleatória dos átomos.
3) Implementar a simulação de um episódio. Um episódio inicia com o tanque cheio e termina depois de n movimentações de átomos. O valor de n deve ser determinado empiricamente.
4) Rodar inúmeras simulações e anotar os locais finais do carro. Com esses dados, produzir o gráfico similar ao que aparece no minuto 7:27. O resultado é o mesmo do vídeo? Se não, por que?
5) Faça variações no valor de n e avalie o resultado.
6) Faça variações nos tamanhos do tanque e do ambiente e avalie o resultado.
7) (bônus) Faça a animação gráfica dos átomos.
Para algumas das funcionalidades acima, sugere-se estudar as bibliotecas random e matplotlib do python (ou outras que acharem relevantes).
O trabalho pode ser feito em equipes de até 3 alunos. Conforme cronograma da disciplina, a equipe deve fazer uma apresentação dos resultados para a turma, entregar um relatório e o código fonte da implementação. Critérios de avaliação do projeto:
- qualidade do código fonte;
- qualidade do relatório; e
- rigor na avaliação dos resultados gerados.
